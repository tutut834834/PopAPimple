# Pimple Pop Precision

A GitHub Pages-ready HTML/CSS/JavaScript mini game.

## Features

- Cartoon male face with 9 pimples
- 3 small, 3 medium, and 3 big pimples
- Mobile two-finger squeeze support
- Desktop mouse squeeze support
- Pus animation on successful pop
- Red wound flash when squeezing too long
- Feedback after every pimple
- Points based on timing, finger position, and pressure
- Combo multiplier
- Fully static, no server needed

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, `game.js`, and `README.md` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. Your game will appear at your GitHub Pages URL.

## Local testing

Open `index.html` in a browser.
